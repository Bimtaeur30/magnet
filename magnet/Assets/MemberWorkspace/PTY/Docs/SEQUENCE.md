## 새 Phase 기록 형식

`Sequence/phaseN.md` — Phase 완료(또는 진행) 시 갱신. 형식은 `Assets/MemberWorkspace/JTH/Docs/Sequence/phase0.md` 참고.

- **목표** — `Docs/DESIGN.md` 완료 기준
- **한 일** — 이 Phase에서 실제로 한 것 / 안 한 것
- **코드·에셋 — 어디를 보면 되나** — 표로 경로 정리
- **메모** — git에 안 남는 함정·결정만
